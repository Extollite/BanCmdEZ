# BanCommands

Simple mod for [ElementZero](https://github.com/Element-0/ElementZero).<br><br>
This mod allows you to ban usage of certain commands for non operators.

Config
-------------
```
  BanCmd:
    enabled: true
    bannedCommands:
      - gamerule
    blockedCommandMessage: You cannot use this command!
    logCommands: true
```
